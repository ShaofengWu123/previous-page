  • 数论初步
    • 整除性、因子
    • 最大公因子
      • 定理2.2：
        • (a,b)是集合S={ax+by|x,y是整数}中最小的正整数，并且ax+by = k(a,b)
        • 整数n可以表示成ax+by形式的充要条件是(a,b)|n，原因是ax+by = k(a,b)
      • (ma,mb) = m(a,b)，m是正整数
      • (a,b)=d,那么(a/d,b/d) = 1
      • a,b的公因子是最大公因子(a,b)的因子
      • (a,m)=(b,m)=1，那么(ab,m)=1
      • (a,b) = (a,b+ax)
      • c|ab且(c,b)=1，那么c|a
      • 辗转相除法：求不定方程解以及最大公因子
        • 求两个数最大公因子：大数对小数辗转相除，最后余数为0时小数为最大公因子
        • 方程右侧为左侧两系数的最大公因子，先进行系数辗转相除，然后进行逆过程
    • 最小公倍数
      • 定理：a，b公倍数是[a,b]的倍数
      • [ma,mb] = m[a,b]，m为正整数
      • 若(a,b)=1，即互素，[a,b] = ab
      • [a,b](a,b)=ab
    • 素因子分解唯一定理：任何正整数有且仅有一种方式写成素数乘积
    • 线性不定方程：ax+by=n
      • 有解条件：(a,b)|n，因为ax+by=k(a,b)
      • 通解：x = x0 + b/(a,b)，y = y0 - a/(a,b)，特解可取x0=0，计算得到y0
    • 同余式和线性同余方程
    • 模n同余、同余式：m|(a-b)，即a与b模n同余
      性质
    • 线性同余方程解法：ax ≡ b(mod m)
      • 有解条件：(a,m)|b，即ax+mk =b，x、k有解要求b是(a,m)的倍数
        当(a,m)=1时，对每个b，都存在x、k，所以一定有解，并且有模m唯一解
      • 如果满足有解条件(a,m)|b，那么同余方程的解为
            x = x0 + m/(a,m) * t mod(m), <=0t<=(a,m)-1
        其中x0是同余方程a/(a,m) x0 ≡ b/(a,m) mod(m/(a,m))的解
      • 解方程过程中要利用同余式性质对方程进行化简
    • 线性同余方程组
      • 中国剩余定理：xi ≡ ai(mod mi)
        • 要求：mi两两互素，ai任意整数，此时一定有解，并且解是模M = m1m2...mr唯一的
        • 步骤
          • 计算M以及各Mi
          • 解方程Mibi ≡ 1(mod mi)得到bi
          • 方程组的解为 y ≡ ΣMibiai (mod M)
      • 解方程过程中一定要注意化成mi彼此互素的形式，并且多使用同余式的本质m|(a-b)或者ax+mk = b，后者通过辗转相除法(当b=(a,m))或者通式的方法可以解(当(a,m)|b时)，有时候通式难以获得特解，就考虑辗转相除法，等于是已知最大公因子，重复辗转相除法的过程，把系数算出来；如果不满足辗转相除法b=(a,m)，那么只能试特解
      • 特殊情况：两个方程组成的线性同余方程组，其有解条件为(m1,m2)|(a1-a2)，并且解是模[m1,m2]同余。对于这种方程，如果m1、m2不互素，不能用中国剩余定理，那么可以考虑令x = a1+m1k1，带入第二个方程，最终化成x = x1+k[m1,m2]，即得结果
    • 欧拉定理和欧拉函数
      • 完系和缩系：完系即从模m同余类每个类中选出一个代表元组成一个集合，而缩系只包括完系中满足(x,m)=1，即与m互素的同余类
        • 注意：如果x，y属于一个模m同余类，并且(x,m)=1，那么一定有(y,m)，即这个同余类里的所有元素都和m互素
        • 欧拉函数φ(m)：m的缩系大小，即与m互素的同余类个数，也即不超过m的与m互素的正整数个数
          • 例子：模6的完系可以写为{0,1,2,3,4,5}，缩系为{1,5}，φ(6)=2
          • 计算公式：
            • 对于素数，φ(p)=p-1
            • p为素数，φ(pm) = pm(1-1/p)
            • 若(m,n)=1，φ(mn) = φ(m)*φ(n)
            • 若n的素因子分解包括p1,...,pk，那么φ(n)=n(1-1/p1)...(1-1/pk)
      • 欧拉定理和费马定理
        • 引理：若(a,m)=1,{x1,...,xm}是模m的完系，那么{ax1,...,axm}也是模m完系；{r1,...,rφ(m)}是模m缩系，那么{ar1,...,arφ(m)}也是模m缩系
        • 欧拉定理：若(a,m)=1,aφ(m)≡1(mod m)
        • 费马定理：取欧拉定理中m为素数p，那么(a,p)=1表示a不能整除p(可能大于p也可能小于)，这时有：ap-1≡1(mod p)；特别的，对于任何a，有ap≡a(mod p)
          • 利用费马定理解题的方法：
            • 若m不是素数，可以对其素因子分解，利用费马定理找到同余关系式,然后关系式相乘等将原方程转化成线性同余方程组，然后使用中国剩余定理
            • 若同余方程的m为素数，ap-1≡1(mod p)，两边可以同时相乘，由于右边是1，所以不会变化
    • 威尔逊定理：判断素数

  • 二元关系：特殊的集合
    • 基本概念
      • 概念：关系R、二元关系、空关系/平凡关系、万有关系
        • 关系R：A1 x A2 x .... An计算得到积集的子集(即一部分有序n数组组成的集合)，称为A1 x A2 x .... An上的一个n元关系R
        • 二元关系：A x B上的二元关系R，也叫A到B的关系
          • A上的二元关系：即A=B的情形
          • 二元关系R的定义域和值域：即出现在有序数组子集内的a和b
          • aRb：表示a与b有关系R，即(a,b)属于R。同理，a、b没有关系R即(a,b)不属于R
      • 关系的性质(对A上关系而言)：自反、不自反、对称、反对陈、传递(p56)
        • 注意：自反和不自反不是互补的，因为两个都要求全部元素满足x和x有关系R/x和x没有关系R，这意味着某关系可以是既不自反也不不自反的
      • 关系的表示：关系矩阵、关系图
      • 关系运算：直接套用集合之间关系
        • 关系相等、小于、大于、小于等于、大于等于
          • 关系R1<=R2：即若xR1y，必有xR2y
        • 合成关系
          • 定义：x(R2○R1)y <=> 存在z属于B，使xR1z，zR2y
            其中x、y分别属于A、C，R1、R2分别是A到B，B到C关系
          • 定理4.1：关系合成没有交换律，但具有结合律
            • 提示：关系相等等价于集合相等
        • 闭包运算
          • 定义：R向R1扩充，且R1具有某种性质(自反性、传递性、对称性...)，并且R1是扩充的最小结果 -> R1是R的该性质闭包
            • R1具有某种性质
            • R是R1的子集
            • 任何一个具有该性质的R2,且R是R2的子集,必然有R1小于等于R2
          • 传递闭包、自反闭包、对称闭包构造方法
            • 传递闭包：aR+b <=> 存在n，使得aRnb
              即对A内元素不断计算R、R2...，最后把所有关系都算出来即可
            • 自反闭包：R U IA即可
            • 对称闭包：把R中所有逆关系都补上即可，即对(x,y)属于R，补充(y,x)
            • 证明：按照闭包的定义的三点进行证明(★★)
    • 等价关系
      • 定义：A上具有自反、对称、传递性质的关系
        • a与b等价即aRb，其中R是等价关系
      • 等价类、划分
        • 等价类
          • 定义：[a],即与a具有等价关系R的全体元素构成的集合，称为a所属的等价类
          • 定理4.5：等价类要么相等，要么不相交，并且所有等价类的并得到A(★★★)
            • 提示：前半部分简单，各等价类的并和A相等利用集合相等的证明方法(集合包含的反对称性)
        • 划分
          • 定义：集合的一个划分{A1,A2,...}
            • Ai是A的非空子集
            • Ai与Aj要么相等,要么不相交
            • 所有Ai的并得到A
          • 划分和等价关系
            • A的等价类集合是A的一个划分，即等价关系产生划分
            • 定理4.6：现有A的一个划分,构造一个关系R，其满足xRy <=> 存在i，x属于Ai且y属于Ai,那么R是一个等价关系,即划分对应等价关系(★★★)
              • 提示：按照等价关系的定义证明
            • 不同等价关系可能产生相同的划分，那么这个时候这两个等价关系是相等的，即划分相同等价于是同一个等价关系
    • 序关系
      • 部分序关系
        • 定义：集合A的部分序关系ρ，A和ρ构成部分序集<A,ρ>
        • 可比较的：即aRb，对于A中任意两个元素,不一定是可比较的
      • 线性序关系
        • 定义：集合A上的部分序关系ρ，且A中任意两个元素都可以比较，即aρb和bρa至少有一个成立，A和ρ构成线性序集<A,ρ>
        • aρ~b：表示aρb且a!=b
      • 极大元和极小元、Hasse图
        • 控制
          • 定义：在部分序集上，aρ~b且不存在z使得aρ~z，zρ~b，那么b控制a，或者说a被b控制，记为aρ○b
            • 即控制关系不允许由传递性推导而来，方便画Hasse图
            • 例子：R <= 组成的部分序集，没有元素控制其他元素
          • 定理4.7：A是有限集合,对于A中元素a,若存在b属于A，使得aρ~b，那么a的控制元素一定存在(★)
            • 提示：紧扣控制的定义
        • 极大元、极小元
          • 定义：对于部分序集，如果不存在元素b使得aρ~b/bρ~a，那么a是部分序集的极大元/极小元
            • 某元素a不是极大元，说明b属于A，满足aρ~b
            • 不存在指的是可能不可比，也可能可比但关系相反
        • Hasse图：上方画极大元、下方画极小元，然后将控制元素和被控制元素相连
      • 最大元和最小元
        • 定义：对A中任意元素y,都有yρ~x/xρ~y，那么x是A的最大/小元
        • 最大元和极大元关系
          • 最大元必是极大元(★)
          • 最大元一定只有一个(★)
          • A存在最大元等价于A的极大元只有一个(★★★)
            • 证明必要性时任取y属于A，即按照最大元定义证明
      • 上界和下界
        • 概念：对于A的某子集M，a属于A，且对任意m属于M，都有mρa，则a是M的上界
        • 性质：
          • 上下界不一定存在，存在也不一定唯一
          • 某子集的上下界不一定属于这个子集
        • 最小上界、最大下界
          • 概念：x是M的上界,对于M的每个上界a，都有xρa，那么x为M的最小上界
    • 集合的势：衡量集合规模
      • 等势：A\B之间存在双射，记为A~B
        • 等势是等价关系，可以按势来划分等价类(元素是集合)，同一等价类的集合等势
        • 通过构造双射比较集合势的大小
      • 有限集合和可数集合
        • 有限集合：和自然数断片等势的集合
        • 无限集合：不是有限集合的集合
        • 可数无限集合：和自然数集合等势的无限集合，势记为N0
        • 实数集合的势记为N1
      • 无限集合
        • 定理4.13：无限集合必有一个可数无限的子集(★)
        • 定理4.14：无限集合一定与其某个真子集等势(★)
          • 提示：从4.13得到一个子集，构造双射
          • 无限集合的定义：与自己的某个真子集等势的集合是无限集合
      • 势的大小
        • 等势：即势相等，集合之间存在双射
        • 支配/小于等于：A与B的一个子集等势,那么B支配A,并且说A的势<=B的势
        • 小于：B支配A,并且A、B不等势，A的势小于B的势，注意：与真子集无关！！！
          • 不能说A与B的真子集等势就是A的势小于B的势，严格按照定义证明小于等于再证明不等势
        • 定理4.11：A的势小于A的幂集的势(★★★)
          • 提示：严格按照定义，小于等于很好证，不等势用反证法，a属于g(a)或不属于，内部成员和外部成员集合B，令g(b)=B，分析b属于B或者不属于B，矛盾
        • 定理4.12：支配关系是部分序关系(略)

          • G中某元素a阶为n，若am=e，那么n|m
          • 定理5.1：群有左消去律和右消去律
          • 定理5.2：群中a*x=b，x的解唯一
          • 定理5.3：群中单位元和逆元唯一
          • 定理5.4：群中元素的逆满足：(a')' = a，(a*b)' = b'*a'
    • 群的等价定义
    • 子群
      • 定义与证明方法：G是群，满足以下条件的H是G的子群，记为H<=G
        • H是G的非空子集
        • 封闭性：任意a、b属于H，a*b属于H
        • 逆元：任意a属于H，a在G中的逆元属于H
      • 性质：
        • 定理5.8：子群是群
        • 定理5.9：如果H是群G的非空有限子群，那么只需要满足封闭性即可确定H是G子群
      • 生成子群
        • 定义：<S>是G的子群中包含集合S的最小子群
        • 等价定义：设A为G子群中包含集合S的所有子群的集合,这些子群的交得到一个群K，此群即为S的生成子群
          • 证明提示：证明K是群，包括非空、封闭、元素逆，由于K是所有包含S子群的交,而K也是群，因此K最小
        • 生成子群构造方法：从集合S中取任意数量的任意元素的本身和逆，进行*运算，即可得到S的生成子群<S>
          • 证明提示：证明这样的集合是一个群，接着证明这样的集合和<S>相互包含
          • 例子：S={a}，<S> = {an|n是任意整数}
    • 循环群
      • 基本概念：每个元素都能写成某元素的幂的群，其中该元素g称为生成元
      • 性质：
        • 定理5.10：若g是G中一个k阶元，那么由g的任意整数次幂构成的集合是G的一个k阶子群;特别的，如果G本身是k阶群，而g是G中k阶元，那么g一定是G的生成元，此时G是一个循环群
          • 意义：k阶群中有k阶元，说明该群是由此元素生成的k阶循环群
        • 定理5.11：循环群的任意子群是循环群(★★★)
          • 提示：假设子群中某元为gn，n的最小值为m，n=um+v，0<=v<m，证明gv也属于子群，从而m最小矛盾
        • 定理5.12：阶为n循环群G的生成元为a，其子群H为以b=as为生成元的群，那么H为n/(n,s)阶循环群
          • 理解：用s去凑n的倍数，需要凑n/(n,s)次
    • 置换群
      • 基本概念：集合A上双射全体对于映射的合成运算构成群，叫做对称群，对称群的子群叫置换群
        • n次对称群：n元集合上全体置换对置换合成构成群Sn，这种群叫做n次对称群
        • 置换不满足交换律，因此对称群、置换群一般是非交换群
    • 群的同构
      • 同构映射：“保持运算”，即存在从G1到G2的双射f，对任意a、b属于G1，满足f(a*b) = f(a)·f(b)
        • 两个群同构，它们之间存在b保持运算双射，但是也可能有双射不满足保持运算
        • 证明两个群同构，即构造保持运算的双射，或者先构造或者证明满同态映射，然后得到模Kerf的商群和另一个群同构
      • 定理5.15：同构映射能保持群的结构，即G为群，H为集合，若存在从G到H的保持运算双射，那么H和符合这一点的运算构成群，并且G与H同构
      • 性质：
        • 任何一个群与一个置换群同构
          • 提示：首先搞清楚置换群的元素是映射，不要把这个映射和同构映射搞混，然后构造一个置换群{f1,f2,...}，其中fi(x) = aix，a、x均属于G，即每个ai对应一个fi，然后证明这是一个置换群，最后证明ai到fi这个双射满足运算保持即可
        • 单位元像是单位元，逆元的像等于像的逆元
  • 商群
    • 陪集与lagrange定理
      • 基本概念：G中两个元素模H同余关系、左/右陪集、指数
        • 模H同余关系R：H是G的子群，如果a,b属于G，且a*b'属于H，那么说a、b有模H同余关系，记为a≡b(mod H)
        • 陪集：模H同余关系是等价关系，并且G中某元素a的模H同余等价类为Ha，称为G中H的右陪集，a是陪集Ha的代表元。左陪集改为a'*b属于H，对应左陪集aH
        • 指数：G关于子群H的左/右陪集个数叫做H在G中的指数，记做[G:H]，也即陪集对应的同余关系将G划分成几块
        • 性质：
          • He=H
          • a、b模H同余等价于a、b属于同一个模H同余等价类等价于Ha=Hb等价于a*b'属于H
          • a属于H等价于Ha=H
          • 定理6.2：H是G子群，H所有左陪集集合和右陪集集合等势
            • 注意：构造双射时首先要说明映射与代表元选取无关，因为一个陪集的代表元有多个
          • 定理6.3：Lagrange定理，G是有限群，H是其子群，有：|G| = [G:H]|H|(★★★)
            推论：抓住同余等价类的获得方法，对任意一个子群H，都可以获得相应的模H同余等价类，必要的时候构造一个特殊的子群H说明问题
            • 模H同余各等价类元素个数相等
            • 群G的元素阶是群阶的因子
            • 素数阶群元素阶为1或p，并且该群一定是循环群
    • 正规子群与商群
      • 基本概念
        • 正规子群：H是G子群，并且对任意g属于G，h属于H，有g'*h*g属于H成立，那么H为G的正规子群，记为H◁G
          • 注意：正规子群定义是对于任意g和h而言的
          • 性质：
            • 定理6.4：H是G的子群，H是G的正规子群等价于对于任意g，H的左陪集Hg等于右陪集gH
            • 指数为2的子群是正规子群
            • 交换群任何子群都是正规子群
        • 商群：正规子群的陪集组成的群
          • 基本概念：若两集合相乘定义为元素按群G的运算相乘，那么<{Ng|g属于G},·>是群，称为G模N的商群，记为G/N
          • 性质：
            • Ng1·Ng2 = N(g1*g2)，并且陪集间乘法运算与代表元选取无关
            • |G/N| = [G:N]，即商群的阶等于指数
            • 交换群的商群依然是交换群
            • G是有限交换群，素数p是|G|的因子，那么群G中必有一个p阶元(★★★)
              • 提示：归纳法，假设阶小于k有p阶元，然后对阶等于k时，任取阶为t的元素a，分p能否整除t来讨论；其中后一种情况考虑<a>的商集，商集也是交换群并且由归纳假设，有p阶元...
        • 同态映射
          • 同态映射：从群G1到G2的运算保持映射，叫做同态映射，如果该映射是满、单、双射，称为满同态映射、单一同态映射、同构映射
          • 核：G1中映射到G2中单位元e2的那些元素组成G1的核Kerf，即对于任意a属于Kerf，f(a) = e2
            性质
            • f是同态映射，那么Kerf是G1的正规子群，可以将G1按照模Kerf划分
            • Kerf = {e1}等价于同态映射f是单射
            • 若G1=Kerf,此映射称为零同态映射
            • 定理6.8：f是同态映射，对任意a属于G1，f-1(f(a)) = aKerf
              • 理解：模Kerf同余类里的元素映射到G2中相同的值
          • 定理6.9：（群同态基本定理）群G1的任何一个商群都是G1的满同态像；如果G2是G1的满同态像(同态映射为f)，那么G1/Kerf和G2同构(★★★)
            • 理解：G1和G1任意商群之间存在自然同态f(a)=aH；G2与G1存在满同态映射，那么用Kerf对G1进行划分，得到的商群和G2是同构的，即把G1中像相同的进行归类，并且各等价类还是保持运算的
            • 定理6.10：H,K是G的正规子群，且K是H的子集，那么 G/H = G/K / H/K
              • 理解：用K划分更细
  • 环和域
    • 环
      • 环定义：交换加群+带1半群乘群+乘法对加法左右分配律即为环
      • 其他概念：交换环(乘法满足交换律)、可逆元(有乘法逆元的元素)、零元、幺元、负元、逆元、平凡环(阶为1的环必为{0}，即平凡环)\非平凡环
      • 性质
        • a · 0 = 0 · a =0
        • a · (-b) = (-a) · b = -(a · b)，(-1)·a = -a
        • -1 · -1 = 1
        • 若0=1，那么R为平凡环/若R的阶为1，那么R为平凡环
        • 环中所有可逆元的集合对乘法运算构成群，其中一定包括1，因为1一定可逆且其逆元是1
    • 整环和域
      • 零因子：对于非零元素a，如果存在另一个非零元素b，使a·b=0，那么a、b是彼此的零因子
        • 注意和零元区分
        • a·b=0且a、b不为零，那么a、b互为彼此的左零因子/右零因子
        • 环中没有左零因子那么也没有右零因子
        • 定理7.2：环中没有零因子等价于环中乘法满足左右消去律
      • 整环：无零因子非平凡交换群
        • 整环有乘法左右消去律
        • 整环中如果1R的加阶无限，那么所有非零元素的加阶无限；如果1R加阶有限，那么必为素数，从而所有非零元加阶也是素数
          • 整环的特征为非零元素加阶，即p；特殊情况下，加阶为无穷，那么特征为0
      • 域：乘法部分为交换群，即加法、乘法都是交换群，并且乘法对加法有分配律
        • 域是整环
        • 有限整环是域
        • 证明域，即证明加法、乘法分别构成交换群，并且乘法对加法左右分配；当乘法逆元存在不好证明时，可以考虑证明有限整环，即没有零因子的非平凡交换环
    • 子环和环同态
      • 子环满足以下条件：正好满足子环是环的条件
        • 加法是子群
        • 乘法封闭性
        • 乘法的单位元在S中(不要求乘法逆元)
      • 环同态映射
        • 定义
          • 加法保持运算
          • 乘法保持运算
          • 乘法单位元映射到乘法单位元
        • 性质
          • 零元映射到零元
          • 负元映射到负元
          • 若有逆元，元素的逆元映射到元素像的逆元
          • 同构映射将整环(域)映射到整环(域)
          • 保持运算的满射(不一定映射单位元)将环映射到环
        • 自同态映射f:G->G，其中<G,+>是交换群，定义新的对于自同态映射的二元运算+和+，得到的<E,+,·>是一个环，称为自同态环
          • (f+g)(x) = f(x) + g(x)
          • (f·g)(x) = f(g(x))
    • 理想和商环
      • 定义：若I是环R子集，对于任意x，y属于I，任意r属于R，都满足以下条件，那么I是R的一个理想。
        • x-y属于I，由此可推出I是R对加法的子群
        • x·r属于I且r·x属于I
      • 平凡理想(R和{0})、真理想(不是平凡理想的理想)
      • 商环：I是R的理想，定义R/I={x+I}，则<R/I,+,·>是环，并称为R模I的商环
        • 求商环：用每个x去加I，得到的集合看是否有相等的，相等的可以归为一类，如果都不相等，那么每个x对应一个x+I
        • 商环中加法满足结合律和交换律，商环的零元是0+I，-x+I是x+I的负元，因此<R/I,+>是交换群
        • 商环中乘法满足结合律，商环的幺元是1+I，<R/I,·>是带1半群
  • 格和布尔代数
    • 格
      • 基本定义：部分序集中任意a、b，{a,b}都有最大下界和最小上界，那么部分序集称为格
        • 积：记为*，最大下界，和：记为⊕，最小上界
      • 等价定义：如果*,⊕是集合上二元运算，并且对任意的a,b,c满足以下三条，那么<A,*,⊕>是格
        • 结合律：(a*b)*c=a*(b*c)，对和同理
        • 交换律：a*b=b*a，a⊕b =b⊕a
        • 吸收律：a*(a⊕b)=a，a⊕(a*b)=a
      • 性质：证明抓住最大\最小，可结合图来提示
        • 对偶关系：在格中成立的命题，将大于等于和小于等于颠倒，将和、积颠倒，命题成立
        • 幂等律：a*a=a,a⊕a=a
        • 交换律：a*b=b*a，a⊕b =b⊕a
        • 结合律：(a*b)*c=a*(b*c)，对和同理
        • 吸收律：a*(a⊕b)=a，a⊕(a*b)=a
        • a<=b等价于a*b=a等价于a⊕b=b
        • 如果b<=c，那么a*b<a*c，a⊕b<a⊕c
        • 分配不不等式：a*(b⊕c) <= (a⊕b)*(a⊕c)
        • 格的任意有限子集有最大下界和最小上界
      • 完全格：对格A的任意子集有最小上界和最大下界
        • 对于A是有限的时候,由上面性质可知A是完全格
      • 有界格
        • 定义：格存在最大元和最小元，记为1和0，称为有界格
        • 性质
          • 任意a属于A，0<a<1
          • a*1=a，a⊕1=1，a*0=0，a⊕0=a
        • 补：a*b=0,a⊕b=1，那么a、b互为补；特殊的，1和0互为唯一的补
        • 有补格：有界格中，如果每个元素都有补，称为有补格
      • 分配格
        • 定义：若任意元素，有和、积对彼此的分配律，则为分配格
          • a*(b⊕c)=(a⊕b)*(a⊕c)
          • a⊕(b*c)=(a*b)⊕(a*c)
        • 性质：
          • 分配律
          • 任意线性序集是分配格
          • 分配格中若a,b,c满足a*c=b*c，a⊕c=b⊕c，那么a=b（使用吸收律和分配律证明）
          • 有界分配格中若a有补元，那么其补元唯一
          • 对于有界分配格，a,b若有补元，有摩根律：(按定义证明即可)
            • (a*b)' = a'⊕b'
            • (a⊕b)' = a'*b'
      • 布尔格：有补分配格
      • 模格：
        • 定义：对任意a,b,c，当a<=b，都有a⊕(b*c)=b*(a⊕c)
        • 每个分配格都是模格
        • A是模格等价于对任意a,b,c，当a<=b且a*c=b*c，a⊕c=b⊕c，有a=b
      • 布尔代数：由布尔格(有补分配格导出的代数系统)
        • 若A至少有两个元素，*和⊕是A上二元运算，且满足以下条件,则称<A,*,⊕,',0,1>是布尔代数
          • 交换律
          • 分配律
          • 0,1属于A，且对任何元素，a*1 = a,a⊕0 = a
          • 对A中任意元素存在补元
        • 性质：有补分配格
          • 每个元素有补元且唯一
          • 分配律、交换律、结合律
          • 若x<=y，那么x*y'=0
          • x<=y等价于y'<x'
